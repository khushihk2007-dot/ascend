# Motivation

A high-end cinematic, luxury, minimalist motivational workspace featuring rich ambient audio and visual systems.

## Tech Stack
- **React**
- **Vite**
- **Tailwind CSS**
- **Motion**

## Getting Started

### Prerequisites
- Node.js

### Installation
1. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally
To start the development server:
```bash
npm run dev
```

### Building for Production
To build the project:
```bash
npm run build
```
